# 🚀 AI Research Studio

An AI-powered research assistant that generates comprehensive, well-structured research reports on any topic using AI and real-time web search.

Built with a modern frontend and an n8n automation backend, AI Research Studio streamlines the research process by combining LLM reasoning with live web data.

---

## ✨ Features

- 🔍 AI-powered research generation
- 🌐 Real-time web search integration
- 📑 Structured research reports
- 📊 Executive summaries

## ⚙️ How It Works

1. User enters a research topic.
2. The frontend sends the request to an n8n webhook.
3. n8n processes the request.
4. DeepSeek generates a research strategy.
5. Tavily performs real-time web research.
6. DeepSeek produces a structured research report.
7. The report is returned to the frontend for display.
